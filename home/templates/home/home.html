¡Bienvenido a API IMC!

Registra tus datos para obtener tu IMC

La API de Registro y Cálculo de Índice de Masa Corporal (IMC) es una potente herramienta diseñada para facilitar el seguimiento de la salud y el bienestar de los usuarios. Permite registrar datos vitales como el nombre del usuario, su peso y su altura, y luego realiza el cálculo del IMC para brindar una medida objetiva de la composición corporal.

Registro de datos

Nombre: {{ nombre }}
Altura: {{ altura }} metros
Peso: {{ peso }} kilogramos 



Endpoint:
POST api/registrar-usuario/

    Content-Type: application/json
    {
    "nombre": "Ejemplo Usuario",
    "altura": 1.75,
    "peso": 70.5
    }

Endpoint para Obtener Usuario por Nombre

El endpoint para obtener un usuario por su nombre proporciona una forma eficiente de recuperar los detalles de un usuario específico utilizando su nombre como identificador. Esta función es útil para acceder a la información de salud de un usuario en particular y realizar consultas personalizadas. 



Endpoint:
GET api/usuarios/{Nombre}

Ejemplo de Respuesta Exitosa:

{
    "id": 123,
    "nombre": "Ejemplo Usuario",
    "altura": 1.75,
    "peso": 70.5,
    "imc": 23.0
}
