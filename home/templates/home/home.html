¡Bienvenido a API IMC!

Registra tus datos para obtener tu IMC

La API de Registro y Cálculo de Índice de Masa Corporal (IMC) es una potente herramienta diseñada para facilitar el seguimiento de la salud y el bienestar de los usuarios. Permite registrar datos vitales como el nombre del usuario, su peso y su altura, y luego realiza el cálculo del IMC para brindar una medida objetiva de la composición corporal.

Registro de datos

Nombre: {{ nombre }}
Altura: {{ altura }} metros
Peso: {{ peso }} kilogramos 



Endpoint:
POST https://userimc.onrender.com/api/registrar-usuario/

    Content-Type: application/json
    {
    "username": "Ejemplo Usuario",
    "altura": 1.75,
    "peso": 70.5
    }

Endpoint para Obtener Usuario por Nombre

El endpoint para obtener un usuario por su nombre proporciona una forma eficiente de recuperar los detalles de un usuario específico utilizando su nombre como identificador. Esta función es útil para acceder a la información de salud de un usuario en particular y realizar consultas personalizadas. 



Endpoint:
GET https://userimc.onrender.com/api/usuarios/{Nombre}

Ejemplo de Respuesta Exitosa:

{
    "id": 123,
    "nombre": "Ejemplo Usuario",
    "altura": 1.75,
    "peso": 70.5,
    "imc": 23.0
}

Endpoint para Obtener Todos los Registros

El endpoint para obtener todos los registros de usuarios brinda una vista integral de todos los perfiles de usuarios almacenados en el sistema. Esta función es valiosa para el análisis agregado y la revisión general de la información de salud de los usuarios. 



Endpoint:
GET api/usuarios/

Ejemplo de Respuesta Exitosa:
[
    {
        "id": 1,
        "nombre": "Usuario1",
        "altura": 1.70,
        "peso": 65.5,
        "imc": 22.6
    },
    {
        "id": 2,
        "nombre": "Usuario2",
        "altura": 1.65,
        "peso": 70.0,
        "imc": 25.7
    },
]

Endpoint para Obtener los Registros por ID

El endpoint para obtener un usuario por su ID proporciona una manera eficaz de acceder a los detalles de un usuario específico utilizando su identificador único. Esta función es útil para obtener información detallada sobre un usuario en particular y realizar operaciones personalizadas. 



Endpoint:
GET https://userimc.onrender.com/api/usuarios/{id}

Ejemplo de Respuesta Exitosa:

{
    "id": 123,
    "nombre": "Ejemplo Usuario",
    "altura": 1.75,
    "peso": 70.5,
    "imc": 23.0
}

Endpoint para Obtener la Estadística de los Regístros

El endpoint para obtener estadísticas de usuarios proporciona una visión general del estado de los usuarios registrados en la plataforma. Esta función ofrece información clave, como el número total de usuarios registrados, así como promedios de IMC, peso y altura. 



Endpoint:
GET https://userimc.onrender.com/api/estadisticas/

Ejemplo de Respuesta Exitosa:
{
    "total_usuarios": 150,
    "promedio_imc": 24.5,
    "promedio_peso": 70.2,
    "promedio_altura": 1.75
}
